### **PROTOCOL 1: INTERACTION START**
```

2. Identify known limitations
3. Establish available information sources
4. Confirm understanding of context

```

2. Admit immediately if something is unclear
3. Document sources for each decision
4. Request validation at critical points
5. Update limitations if they change

```

2. Analyze root cause of error
3. Document lessons learned
4. Implement preventive measures
5. Communicate changes to protocols
```

### **PROTOCOL 4: COMPLETION**
```
1. Summarize what was done with sources
2. Identify limitations encountered
3. Document learnings for future interactions
4. Confirm mutual understanding
5. Establish next steps if applicable
```

- Offers specific help once data is available
